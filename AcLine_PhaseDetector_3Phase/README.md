## Isolated 3-Phase Detector Using Optocoupler

### Features, v4.0
- **Isolation Type:** Opto
- **Isolation IC:** LTV814 x3
- **Display:** LED
- **Output Rule:** Output is AND with all three phases
- **Power Supply:** 3.3V
- **Active Level:** Low
